# AndDream 3D — 品牌沉浸式 3D 体验

AndDream 品牌的交互式 3D 网页。深色空间、液态水银球体、粒子系统与环绕文字带，滚动控制镜头在场景中穿行。

---

## 预览

- 中央巨型磨砂液态金属球体，伴随极强的流动形变动画
- 环绕赤道的荧光文字带：**ANDREAM** · The Dream That Continues
- 340 颗小型水银球体散布在周围空间
- 500 颗粒子液滴与中心球体互动（飞溅、轨道运行、重力回落）
- 11,000 颗背景粒子，冷暖双层分布
- 7 色环形布光（红、橙、金、绿、蓝、紫、粉），各角度呈现不同彩色光泽
- 滚动驱动镜头环绕，自动缓慢漂移，无限循环

---

## 打开方式

直接用浏览器打开 `index.html` 即可，无需任何构建工具或服务器。

> 需要联网：Google Fonts 和 Three.js CDN 需能访问。推荐使用 Chrome 或 Edge。

---

## 操作

| 操作 | 效果 |
|------|------|
| **滚动鼠标 / 触控板** | 镜头在 3D 空间中环绕 |
| **触摸滑动**（移动端） | 同上 |
| **不操作** | 镜头自动缓慢漂移 |
